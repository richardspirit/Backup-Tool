Backup-tool

This is a simple straightforward tool that takes advantage of StoreOnce Catalyst to create and store backups of Oracle and SQL Server databases.

View project on GitHub

Welcome NW Tech.

We are offering a simple, straightforward backup tool that works directly with HP StoreOnce storage using Catalyst. This started as simple project because I got tired of waiting for HP to get back with me about problems that I was having with Data Protecter.

Design

There is less stress on making application look pretty and more on funtionality. The design is simple, straightforward and intuitive. Since this is still in the alpha stages of development GUI will have many inconsistencies and errors but these will be taken care of in later releases after functionality is rock solid.

Backups

The Catalyst software has a CLI interface that we have simply wrapped in an easy to use GUI. All available options will be available as they are available to the Catalyst CLI. Since StoreOnce uses advanced deduplication, compression will have a low priority in the development process. Catalyst uses the native database tools to perform backups so all levels of backups will be available.

Restore

Our tool will list available backups for selected database if any are available.

Development

There are a lot of changes that are to be implemented including a linux version. I want to be able to provide a tool that will take full advantage of the catalyst and StoreOnce technologies while maintaining a simple interface.
